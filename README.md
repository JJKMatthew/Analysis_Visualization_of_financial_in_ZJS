# 浙江省财政情况分析及可视化课程设计
## 项目概述
本项目旨在分析浙江省的财政数据，探索其收入与支出的结构和变化趋势，并通过可视化手段展示分析结果。通过对财政数据的深入挖掘，帮助理解浙江省财政状况，为政策制定提供参考依据。
## 数据来源
数据主要来源于浙江省统计局和浙江省财政厅发布的年度财政报告和公开的财政数据集，涵盖了近十年的财政收入与支出数据。
## 分析内容
1. 财政收入分析
   - 收入结构分析：分析不同来源的财政收入占比，如税收收入、非税收入等。
   - 收入趋势分析：探讨财政收入的年度变化趋势，识别增长或下降的关键年份。
2. 财政支出分析
   - 支出结构分析：分析不同领域的财政支出占比，如教育、医疗、基础设施等。
   - 支出趋势分析：探讨财政支出的年度变化趋势，识别主要支出增长领域。
## 可视化展示
- 使用折线图展示财政收入和支出的年度变化趋势。
- 使用饼图展示财政收入和支出的结构占比。
- 使用柱状图比较不同年份的财政收入与支出。

- 数据可视化：Python（Altair）
- 开发环境：Jupyter Notebook
- 版本控制：Git

- 报告撰写：Markdown
